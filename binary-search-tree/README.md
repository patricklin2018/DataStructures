# 二分搜索树

### 简介

#### 二分搜索树的优势

| # | 查找元素 | 插入元素 | 删除元素 |
|---|--|---|---|
| 普通数组 | O(n) | O(n) | O(n) |
| 有序数组 | O(logn) | O(n) | O(n) |
| 二分搜索树 | O(logn) | O(logn) | O(logn) |
